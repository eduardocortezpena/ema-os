# PROJECT MEMORY

## Decisiones importantes

-   EMA OS será para un único usuario.
-   SQLite fue elegido por simplicidad y portabilidad.
-   Prisma será el ORM definitivo.
-   No convertir EMA OS en ERP.
-   No integrar IA antes de terminar el MVP.
-   UI inspirada en Linear, Notion y Vercel.
-   Sidebar permanente.
-   Server Actions en lugar de APIs innecesarias.
-   Trabajar por sprints.

## Lecciones aprendidas

-   No dejar que Claude llegue a \>80% de contexto.
-   Verificar siempre el resultado ejecutando npm run dev.
-   Auto Mode puede bloquear escritura en `.claude`.
-   Verificar el estado real del proyecto antes de asumir que una tarea
    terminó.
